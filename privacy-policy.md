# Privacy Policy

Last updated: 2026-04-17

## Summary

This extension adds a voice recording button to Microsoft Teams Web. Audio processing happens locally in the user's browser.

## Data handled

- Audio captured from the microphone, only after an explicit user action.
- Microsoft Teams Web interface elements required to find the active composer and attach the generated file.

## How data is used

- Audio is recorded locally in the browser.
- Audio is converted locally into a temporary WAV file.
- The file is only sent to Microsoft Teams when the user stops the recording and triggers the normal attachment flow in the Teams interface.

## What the extension does not do

- It does not send data to developer-owned servers or third parties outside Microsoft Teams.
- It does not collect telemetry, analytics, or user identifiers.
- It does not keep a permanent history of recordings outside the active browser session.
- It does not share data with advertisers or tracking networks.

## Retention

Audio data stays in memory only during recording and attachment preparation. After the attachment flow finishes, the extension clears its in-memory buffers for that session.

## Permissions used

- Access to `teams.microsoft.com` to integrate with the Microsoft Teams Web interface.
- Microphone access, granted by the browser, for local audio recording.
