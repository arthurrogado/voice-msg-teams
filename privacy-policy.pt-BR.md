# Politica de Privacidade

Ultima atualizacao: 17/04/2026

## Resumo

Esta extensao foi criada para adicionar um botao de gravacao de audio ao Microsoft Teams Web. O processamento do audio ocorre localmente no navegador do usuario.

## Dados tratados

- Audio capturado do microfone, somente apos acao explicita do usuario.
- Elementos da interface do Microsoft Teams Web necessarios para localizar o composer ativo e anexar o arquivo gerado.

## Como os dados sao usados

- O audio e gravado localmente no navegador.
- O audio e convertido localmente para um arquivo WAV temporario.
- O arquivo so e enviado ao Microsoft Teams quando o usuario encerra a gravacao e confirma o fluxo normal de anexo pela propria interface do Teams.

## O que a extensao nao faz

- Nao envia dados para servidores proprios ou de terceiros fora do Microsoft Teams.
- Nao coleta telemetria, analytics ou identificadores de usuario.
- Nao armazena historico permanente das gravacoes fora da sessao ativa do navegador.
- Nao compartilha dados com anunciantes ou redes de rastreamento.

## Retencao

Os dados de audio permanecem apenas em memoria durante a gravacao e o preparo do anexo. Depois do fluxo de anexo, a extensao limpa os buffers internos usados na sessao.

## Permissoes usadas

- Acesso ao host `teams.microsoft.com` para integrar a interface do Teams Web.
- Acesso ao microfone, concedido pelo navegador, para gravacao de audio local.
